## What I'm building

### member-360-agentic-experience-layer

A prototype that uses AI agents to navigate healthcare claims data on behalf of members. Instead of scripted chatbot flows, agents reason over 835 EDI transactions, denial codes, and member records to actually explain what happened with a claim and what to do next. Built with Python, LangChain, Streamlit, and React.

### fitness-intel

I ride bikes. This pulls my ride data from Strava, recovery data from Oura, and weather from Open-Meteo into one place. I ask questions in plain English and get answers grounded in real power and recovery numbers — not generic advice. TypeScript, Supabase, Telegram as the interface. Still private while I get the core right.

### cadence

An AI assistant for short-term rental companies. Early stage — more details coming soon.

### github-traffic-tracker

GitHub's traffic API only keeps 14 days of data. This archives it. Zero dependencies, daily cron, JSON archive, static dashboard. Small tool, done properly.
